Sobre o Projeto - Galeria de Imagens com Javascript

Este projeto é uma galeria de imagens criada com Javascript, HTML e CSS, permitindo que os usuários pesquisem e visualizem diversas imagens de alta qualidade. A galeria utiliza a API do Pexels, uma plataforma que oferece uma vasta coleção de fotos e vídeos gratuitos para uso pessoal e comercial.

Principais Recursos:

  1. Pesquisa de Imagens: A galeria oferece uma caixa de pesquisa, permitindo que os usuários pesquisem imagens digitando palavras-chave relevantes. As imagens relacionadas à pesquisa são exibidas automaticamente à medida que o usuário digita.
  2. Exibição em Lightbox: Ao clicar em uma imagem da galeria, uma lightbox é exibida, mostrando a imagem em tamanho maior, juntamente com o nome do fotógrafo. Na lightbox, é possível baixar a imagem clicando no botão correspondente.
  3. Carregamento de Mais Imagens: A galeria carrega inicialmente um conjunto de imagens selecionadas (curated). Quando o usuário chega ao final da galeria, pode carregar mais imagens clicando no botão "Carregar mais", possibilitando a visualização contínua de imagens diversas.
  4. Layout Responsivo: A galeria possui um design responsivo, adaptando-se automaticamente a diferentes tamanhos de tela e dispositivos, proporcionando uma experiência de usuário agradável em qualquer dispositivo.

O projeto utiliza as bibliotecas Unicons para os ícones, bem como a fonte Inter do Google Fonts para garantir uma experiência visual atrativa e coesa.

O código Javascript é responsável por interagir com a API do Pexels, buscar e carregar as imagens, além de criar os elementos HTML para exibição das imagens na galeria. Além disso, a interação com a lightbox é implementada para oferecer uma visualização mais detalhada e opção de download das imagens.

Este projeto é uma ótima maneira de aprender e praticar habilidades de desenvolvimento web, especialmente no que diz respeito à manipulação de APIs, criação de layouts responsivos e interações dinâmicas com o DOM. É uma aplicação útil para quem precisa encontrar e visualizar belas imagens rapidamente.

Você pode acessar este projeto ao abrir o arquivo "index.html" em seu navegador. Experimente pesquisar diferentes termos e explore a galeria de imagens disponíveis!

Observação: Certifique-se de obter uma chave de API válida do Pexels para garantir o funcionamento correto do projeto e evitar problemas de acesso à API.
